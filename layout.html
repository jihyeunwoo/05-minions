레이아웃 실습I

블럭 vs 인라인

가로에 하나의 컨텐츠를 배치하는 HTML 요소의 스타일 기본 속성을 블럭(block level element)이라고 함

인라인 요소(Inline Level Element)는 가로에 여러개의 컨텐츠 요소를 배치하고 컨텐츠가 넘치면 자동으로 줄바꿈 합니다.

레이아웃 실습 II

포지셔닝

컨텐츠의 위치를 지정하는 방법으로 크게 5가지의 속성을 가질 수 있다.

  • position: static [일반적인 흐름에 따라 요소를 배치]
  • position: absolute [절대적인 위치에 배치]
  • position: relative [상대적인 위치에 배치]
  • position: fixed [고정적인 위치에 배치]
  • position: sticky(아직 표준 X [static + absolute])
